Questions and Answers in PHP

  1. what is overloading and overriding in php

    Overloading: Overloading in PHP allows a class to define multiple methods with the same name but different parameters.

    class Example {
    public function overloadedMethod($param1) {
        // Code for method with one parameter
    }

    public function overloadedMethod($param1, $param2) {
        // Code for method with two parameters
    }
}

    Overriding: Overriding in PHP occurs when a child class provides a specific implementation for a method that is already defined in its parent class.

    class ParentClass {
    public function exampleMethod() {
        echo "Parent class method";
    }
}

class ChildClass extends ParentClass {
    public function exampleMethod() {
        echo "Child class method";
    }
}
  2. What is static function in php?

    In PHP, a static function is a function that belongs to a class rather than an instance of the class. It can be called on the class itself without creating an instance of the class. 

    class MathOperations {
    public static function add($a, $b) {
        return $a + $b;
    }
}

$result = MathOperations::add(5, 3);
echo $result;  // Outputs: 8
  3. what is public ,private and protected in oops ?

    In object-oriented programming (OOP), public, private, and protected are access modifiers that define the visibility and accessibility of class members

    Public: Members are accessible from anywhere.

    class MyClass {
    public $publicProperty = "I'm public!";

    public function publicMethod() {
        echo "This is a public method.";
    }
}

$obj = new MyClass();
echo $obj->publicProperty;   // Accessing public property
$obj->publicMethod();        // Calling public method

    private:Members are only accessible within the defining class.

    class MyClass {
    private $privateProperty = "I'm private!";

    private function privateMethod() {
        echo "This is a private method.";
    }
}

$obj = new MyClass();
// The following lines would result in an error
// echo $obj->privateProperty;
// $obj->privateMethod();

    protected: Members are accessible within the defining class and its derived 

    class MyClass {
    protected $protectedProperty = "I'm protected!";

    protected function protectedMethod() {
        echo "This is a protected method.";
    }
}

class DerivedClass extends MyClass {
    public function accessProtected() {
        echo $this->protectedProperty;  // Accessing protected property from a derived class
        $this->protectedMethod();       // Calling protected method from a derived class
    }
}

$obj = new DerivedClass();
$obj->accessProtected();  // Calls accessProtected, which accesses protected members
